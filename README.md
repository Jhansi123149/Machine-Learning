# Machine Learning Projects 🤖

## About
My Machine Learning journey - learning and practicing ML algorithms step by step.

## Projects

### 1. K-Means Clustering (kmeans.py)
- Grouped 300 data points into 4 clusters
- Used Elbow Method to find optimal clusters
- Visualized clusters with different colors

### 2. Customer Segmentation (customers.py, customers2.py)
- Segmented 200 customers by Annual Income and Spending Score
- Used Label Encoding for Gender column
- Identified high income low spending customers for targeting

### 3. KNN Classification (knn.py)
- Classified Iris flowers into 3 categories
- Achieved 96.67% accuracy
- Found best K value using accuracy graph

## Technologies Used
- Python
- Scikit-Learn
- Matplotlib
- Seaborn
- Pandas

## How to Run
1. Clone this repository
2. Create virtual environment: `python -m venv venv`
3. Activate: `venv\Scripts\activate`
4. Install libraries: `pip install scikit-learn matplotlib seaborn numpy pandas`
5. Run any file: `python kmeans.py`
